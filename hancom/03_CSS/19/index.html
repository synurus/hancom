1. 마음에 드는 웹사이트 분석

분석한 사이트

사이트 이름 Obys® Experiment Space — 우크라이나 디자인 스튜디오 Obys의 미공개 작업·실험 아카이브
주소 https://experiment.obys.agency/
마음에 든 이유 스크롤을 내리면 사진들이 링·꽃·구(Sphere)·원통(Cylinder) 모양에서 스르륵 한 줄로 펴지는 3D 인터랙션이 인상적이었다. 흑백 위주의 미니멀한 디자인, 커스텀 커서와 마우스 움직임에 따른 시야각 변경 등 디테일이 뛰어났다.
코드 분석 결과 (클로드야 고맙다!)
HTML 구조 분석
  • <head>에 charset="UTF-8", viewport, og:·twitter: 메타태그(SNS 공유용), 파비콘, 외부 CSS <link>가 정석대로 들어 있다.
  • <body>는 시맨틱 태그 대신 div + class 중심 구조 (Webflow 제작 사이트의 특징): 배경 <canvas> → 3D 갤러리 #obys-fluid-gallery → .page-wrapper(제목 텍스트) → .preloader(로딩 화면) → .mobile-blocker(모바일 안내) 순서.
  • 화면의 핵심이 이미지 태그가 아니라 Three.js가 그리는 <canvas> 한 장이라는 점이 신기했다.
CSS 특징 분석
  • @font-face로 전용 글꼴 "OTF Obys NG"를 직접 등록. letter-spacing이 41번 쓰일 만큼 타이포그래피 중심.
  • 레이아웃은 flex와 grid를 함께 사용하고, 겹치는 요소는 position: relative / absolute / fixed로 배치.
  • transform·transition·animation으로 움직임 처리, 커스텀 커서는 ::before / ::after 가상요소로 십자(+) 모양을 만든다.
  • 반응형은 @media 479 / 767 / 991px 구간 — 우리가 배운 것과 같은 방식!
인터랙션 원리
(JS/Three.js)		 모든 이미지가 Sphere·Ring·Flower·Cylinder·일렬(Horizontal) 5가지 목표 좌표를 미리 계산해 갖고 있고, 마우스 휠로 쌓이는 스크롤 값 하나를 진행도로 삼아 두 좌표 사이를 lerp(보간)로 부드럽게 섞는다. 스크롤 0~1 구간은 "모양 → 일렬 변신", 1 이상부터는 카메라가 옆으로 이동하며 일렬 사진을 훑어보는 아카이브 모드. GSAP 라이브러리로 세부 애니메이션을 다듬었다.
배운 점 화려해 보여도 뼈대는 우리가 배운 HTML 구조 + flex/grid + position + @media 그대로였다. "목표 지점을 미리 정해 두고 진행도 값 하나로 부드럽게 전환한다"는 패턴은 CSS transition의 원리와도 닮아 있어서, 기초가 곧 응용으로 이어진다는 걸 느꼈다.

2. HTML 정리

기본 구조

  1. <!DOCTYPE html> — "이건 HTML 문서야" 선언
  2. <html lang="ko"> — 문서 시작, 한국어 페이지
  3. <head> — 화면엔 안 보이는 준비 칸
  4. <body> — 화면에 보이는 진짜 내용 칸

텍스트 태그

  • 제목 <h1>~<h6> — 숫자가 클수록 글자가 작아짐
  • 문단 <p>, 줄바꿈 <br>, 구분선 <hr>
  • strong은 굵게, em은 기울임, mark는 형광펜

링크와 미디어

  • 링크 <a href="주소" target="_blank">
  • 이미지 <img src="파일" alt="설명">
  • 영상·지도 삽입 <iframe>, 재생 <video> <audio>

목록과 표

  • <ul> 순서 없는 목록 / <ol> 순서 있는 목록 + <li>
  • 표: <table> > <tr> > <th>·<td>
  • 칸 합치기: 가로 colspan, 세로 rowspan

폼(입력)

  • <form> 안에 <label> + <input> + <button>
  • input type: text, email, password, checkbox, radio, date …

Class와 id, 구조 태그

  • class — 여러 요소가 공유 가능 → CSS에서 .이름
  • id — 페이지에 딱 하나 → CSS에서 #이름
  • 시맨틱 태그: header nav main section article footer
  • 범용 상자: <div>(블록) / <span>(인라인)

3. CSS 정리

기본 문법 & 연결

선택자 { 속성: 값; }
              h1 { color: navy; }
  • 외부 <link> (권장) / 내부 <style> / 인라인 style=""

선택자

  • 요소 p → 클래스 .name → 아이디 #name
  • 자손 선택자 li em
  • 구체적일수록, 나중에 쓸수록 우선 적용!

박스 모델

  • 안쪽 여백 padding → 테두리 border → 바깥 여백 margin
  • box-sizing: border-box로 크기 계산 방식 표준화

레이아웃

  • display: flex — 자식을 가로로 나란히
  • display: grid + repeat(3, 1fr) — 행·열 배치
  • position: static / relative / absolute / fixed / sticky

반응형 & 가상클래스

@media (max-width: 720px) {
          /* 폭 조건 만족 시에만 안쪽 규칙 적용 */ }
  • :hover :active (순서는 LVHA)
  • ::before ::after + content 속성 필수

효과 · 변수 · 리셋

  • transform + transition — 카드 animation
  • @keyframes + animation — Title 연필 animation
  • box-shadow, border-radius, linear-gradient
  • CSS 변수: :root { --잉크색: … } → var(--잉크색)
  • CSS 리셋: * { margin:0; padding:0; box-sizing:border-box; }
