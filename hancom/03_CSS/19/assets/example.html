2. HTML 핵심 정리

HTML은 웹페이지의 뼈대(구조)를 만드는 언어입니다.

🏗️ 기본 구조

  1. <!DOCTYPE html> — HTML5 문서 선언
  2. <html lang="ko"> — 문서 전체를 감쌈
  3. <head> — 안 보이는 설정 (charset, title)
  4. <body> — 화면에 보이는 내용

📝 텍스트 태그

  • 제목 <h1>~<h6> — 숫자가 클수록 작아짐
  • 문단 <p>, 줄바꿈 <br>, 구분선 <hr>
  • strong은 굵게, em은 기울임, mark는 형광펜

🔗 링크와 미디어

  • 링크 <a href="주소" target="_blank">
  • 이미지 <img src="파일" alt="설명">
  • 영상·지도 삽입 <iframe>, 재생 <video> <audio>

📋 목록과 표

  • <ul> 순서 없는 목록 / <ol> 순서 있는 목록 + <li>
  • 표: <table> > <tr> > <th>·<td>
  • 칸 합치기: 가로 colspan, 세로 rowspan

⌨️ 폼(입력)

  • <form> 안에 <label> + <input> + <button>
  • input type: text, email, password, checkbox, radio, date …
  • 👉 아래 퀴즈가 radio 입력으로 만들어져 있어요!

🏷️ class와 id, 구조 태그

  • class — 여러 요소가 공유 가능 → CSS에서 .이름
  • id — 페이지에 딱 하나 → CSS에서 #이름
  • 시맨틱 태그: header nav main section article footer
  • 범용 상자: <div>(블록) / <span>(인라인)

3. CSS 핵심 정리

CSS는 웹페이지에 옷을 입히는(디자인) 언어입니다.

✒️ 기본 문법 & 연결

선택자 { 속성: 값; }
h1 { color: navy; }
  • 외부 <link> (권장) / 내부 <style> / 인라인 style=""

🎯 선택자

  • 요소 p → 클래스 .name → 아이디 #name
  • 자손 선택자 li em
  • 구체적일수록, 나중에 쓸수록 우선 적용!

📦 박스 모델

  • 안쪽 여백 padding → 테두리 border → 바깥 여백 margin
  • box-sizing: border-box로 크기 계산 깔끔하게

🧱 레이아웃

  • display: flex — 한 방향 배치 (이 카드들의 부모는 grid!)
  • display: grid + repeat(3, 1fr) — 행·열 배치
  • position: static / relative / absolute / fixed / sticky — 위의 탭 메뉴가 sticky예요

📱 반응형 & 가상클래스

@media (max-width: 720px) {
  /* 좁은 화면용 스타일 */
}
  • :hover :active (순서는 LVHA)
  • ::before ::after + content 필수

✨ 효과 · 변수 · 리셋

  • transform + transition — 카드에 마우스를 올려 보세요!
  • @keyframes + animation — 위의 연필이 흔들리는 이유
  • box-shadow, border-radius, linear-gradient
  • CSS 변수: :root { --잉크색: … } → var(--잉크색)
  • 리셋: * { margin:0; padding:0; box-sizing:border-box; }
