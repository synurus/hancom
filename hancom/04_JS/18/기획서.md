## 5. 핵심 기능 (과제 도전 조건 충족)

### 기능 ① 픽셀 별·꽃 동적 생성 + 스크롤 등장 애니메이션 ⭐
- JS로 배경 별 수십 개를 랜덤 위치·색·크기로 생성 (`createElement`, 반복문, `Math.random`)
- 바닥 픽셀 꽃을 섹션 색에 맞춰 배치
- `IntersectionObserver`로 섹션이 화면에 들어올 때 페이드·슬라이드 인

### 기능 ② 스킨 액자 3D 뒤집기 + 클릭 → 영상 이동 ⭐메인
- 캐릭터 데이터를 **클래스(`class Character`)** 로 정의하고 `new`로 인스턴스 배열 관리 (JS STEP15)
- hover 시 `rotateY(180deg)`로 **3D 뒤집기** → 뒷면에 캐릭터 설명 (CSS STEP08: preserve-3d/backface-visibility)
- 액자 클릭 시 해당 영상 URL을 새 탭으로 열기 (`addEventListener`, `window.open`)

### 기능 ③ 방명록 폼 (form + 배열 + localStorage) ⭐
- 닉네임(`input`) · 별점(`select`) · 메시지(`textarea`) 입력 폼 (HTML STEP15·24)
- 제출 시 `.value`로 값 읽어 배열에 `push` → `map`으로 목록 렌더 → localStorage 저장
- **글자수 카운터**(input 이벤트), **평균 별점**(`reduce`), **최신순 정렬**(`sort`)
- 삭제 버튼은 배열 `filter`로 제거 (JS STEP13)
- 새로고침해도 유지 (localStorage, STEP10)

### 기능 ④ 방문 기록 저장 (localStorage)
- 마지막으로 본 섹션 + "이미 본 스킨" 체크 + 테마 선택을 localStorage에 저장

### 추가 연출·편의 기능
- **밤하늘 다크모드 토글** — `body`에 클래스 토글로 CSS 변수 반전, 선택 저장(localStorage)
- **배경음악** — `<audio autoplay loop>` 자동재생·반복 + 음소거 토글 버튼
- **별똥별 애니메이션** — 랜덤 주기로 생성, Web Animations API(`element.animate`)로 이동
- **스크롤 진행 바 + 맨 위로 버튼** — scroll 이벤트로 진행률 계산·표시
- **커스텀 픽셀 커서** — 기본/클릭 상태별 SVG 커서

> ✅ 위 기능으로 도전 조건(**기능 2개 이상 · localStorage · 반응형**)을 모두 충족.
> ✅ HTML(form/input/select/textarea/nav/audio) · CSS(3D transform/sticky/커서) · JS(class/filter/map/reduce/sort/.value)까지 커버리지 확대.

## 6. 사용 기술 정리 (배운 것 매핑)

**HTML** — 시맨틱 태그(header/nav/main/section/footer), 이미지, 유튜브 `<iframe>`, **폼(form/input/select/textarea)**, **오디오(`<audio>`)**, 버튼, 링크(`target="_blank"`), class/id
**CSS** — 변수(:root/var), flex·grid, `position`(fixed·**sticky**)·z-index, `box-shadow`(픽셀 스프라이트), 그라디언트, `@keyframes`(별 반짝임·둥실·깜빡임), **3D transform**(rotateY·preserve-3d·backface-visibility), `transition`, `::before/::after`(content), **커스텀 커서(`cursor: url()`)**, `image-rendering: pixelated`, `:hover`, `@media` 반응형
**JS** — 변수, 배열·객체, **클래스(class/constructor/new/this)**, 반복문, `Math.random`, 배열 메서드(push·**map**·**filter**·**reduce**·**sort**·includes), 이벤트(click/input/submit/scroll), 폼 입력(`.value`), DOM 조작(createElement·appendChild·innerHTML·classList), 오디오 제어(play/muted), localStorage

**추가 학습(CDN·심화)** — Galmuri 한글 픽셀폰트, `IntersectionObserver`, Web Animations API(`element.animate`)

## 8. 개발 순서 (작게 시작 → 확장)

1. `index.html` — 타이틀 + 5섹션 뼈대부터
2. `styles/main.css` — 흑백·픽셀폰트 → 레이아웃 → 액자 → 별·꽃 → 3D 뒤집기·sticky 네비 → 다크모드·커서·스크롤 UI
3. `scripts/main.js` — 별·별똥별 → 꽃 → 입장 → 스킨 갤러리·필터 → 스크롤 등장 → 방명록 → 다크모드 → 음악 → 진행바/맨위로
4. F12 콘솔로 디버깅 (`console.log`)
5. `@media`로 모바일(1열) 점검
6. **로컬 서버(Live Server 등)로 실행** — 유튜브 임베드·배경음악은 `file://`에서 막히므로(오류 153) 반드시 서버로 확인
7. Netlify 또는 Vercel 배포 → URL 제출
