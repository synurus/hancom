# 백틱(`)과 작은따옴표(')의 차이

JavaScript 기준 정리. HTML/CSS 다음에 JS를 배우면 바로 만나는 내용이다.

## `'` 작은따옴표 — 일반 문자열

```js
const name = '철수';
const msg = '안녕하세요, ' + name + '님';  // 변수를 넣으려면 + 로 연결
```

- 그냥 텍스트를 담는 용도. `"` 큰따옴표와 기능이 완전히 같다.
- 여러 줄을 쓸 수 없고, 변수를 넣으려면 `+`로 이어 붙여야 한다.

## `` ` `` 백틱 — 템플릿 리터럴 (ES6)

```js
const name = '철수';
const msg = `안녕하세요, ${name}님`;  // ${} 안에 변수/식을 바로 넣음
```

백틱만 할 수 있는 것 두 가지:

### 1. `${}` 문자열 보간

변수나 계산식을 문자열 안에 직접 삽입할 수 있다.

```js
`합계: ${price * count}원`
```

### 2. 줄바꿈 그대로 유지 — 여러 줄 문자열

```js
const html = `
  <div class="card">
    <p>${name}</p>
  </div>
`;
```

특히 JS로 HTML 조각을 만들 때 아주 유용하다.

## 정리

|            | `'문자열'`        | `` `문자열` ``     |
| ---------- | ----------------- | ------------------ |
| 변수 삽입  | `+` 연결 필요     | `${변수}` 바로 사용 |
| 여러 줄    | 불가 (`\n` 필요)  | 그대로 가능        |

## 참고

- 키보드 위치: `'`는 Enter 옆, `` ` ``는 숫자 1 왼쪽(한/영 전환키 위).
- 스타일 관례: 변수 없는 단순 문자열엔 `'`, 변수가 들어가면 `` ` ``를 쓰는 게 일반적이다.
