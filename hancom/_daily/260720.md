클래스 이름 -> 인덱스 변환
target_names = ["dining table", "vase"]
name_to_id = {name: idx for idx, name in model.names.items()}
target_ids = [name_to_id[name] for name in target_names]

name_to_id = {name: idx for idx, name in model.names.items()} 코드 설명

model.names는 {인덱스: 클래스이름} 형태의 딕셔너리야. 예:
{0: 'person', 1: 'bicycle', ..., 60: 'dining table', ..., 75: 'vase', ...}
이 줄은 그걸 키와 값을 뒤바꿔서 {클래스이름: 인덱스} 형태로 만드는 딕셔너리 컴프리헨션이야.

name_to_id = {name: idx for idx, name in model.names.items()}
model.names.items() → (idx, name) 쌍들을 하나씩 꺼냄 (예: (60, 'dining table'), (75, 'vase'))
for idx, name in ... → 각 쌍을 idx, name 변수로 풀어줌
{name: idx for ...} → 반복하면서 name을 키로, idx를 값으로 하는 새 딕셔너리를 만듦
결과적으로:

name_to_id = {'person': 0, 'bicycle': 1, ..., 'dining table': 60, 'vase': 75, ...}
이렇게 뒤집어 놔야 name_to_id["dining table"]처럼 이름으로 인덱스를 조회할 수 있어. 원래 model.names는 인덱스로 이름을 찾는 방향(정방향)이라, 반대 방향 조회를 위해 역딕셔너리를 만든 거야.
