Pollinations.ai 로 텍스트 투 이미지 생성
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
[Pollinations.ai](https://pollinations.ai)는 **별도의 회원가입이나 API 키 발급 없이 완전히 무료로 사용할 수 있는 이미지 생성 서비스**입니다.

HTTP GET 요청(URL 방식)으로 호출할 수 있어서 파이썬 코드 몇 줄만으로 즉시 이미지를 만들어낼 수 있습니다.

---

## 🛠️ 파이썬 사용 방법

### 1. `requests` 라이브러리로 이미지 다운로드

가장 흔히 쓰이는 기본적인 방법입니다.

```python
import requests
from urllib.parse import quote

# 1. 한글 또는 공백이 포함된 프롬프트를 URL용 문자열로 변환
prompt = "A futuristic cyberpunk city with neon lights and flying cars"
encoded_prompt = quote(prompt)

# 2. Pollinations.ai URL 생성
url = f"https://image.pollinations.ai/prompt/{encoded_prompt}"

print("이미지 생성 중...")
response = requests.get(url)

# 3. 이미지 파일로 저장
if response.status_code == 200:
    with open("pollinations_output.png", "wb") as f:
        f.write(response.content)
    print("이미지 저장 완료: pollinations_output.png")
else:
    print("에러 발생:", response.status_code)

```

---

### 2. 주요 옵션 활용하기 (파라미터 설정)

URL 뒤에 쿼리 스트링(`?key=value`)을 붙이면 해상도, 시드값(Seed), 이미지 생성 모델 등을 손쉽게 변경할 수 있습니다.

```python
import requests
from urllib.parse import quote

prompt = "A cute red panda wearing glasses, digital art"
encoded_prompt = quote(prompt)

# 옵션 설정
width = 1024       # 너비
height = 1024      # 높이
seed = 42          # 고정 시드값 (동일한 이미지 재현용)
model = "flux"     # 모델 지정 (flux, turbo 등)
nologo = "true"    # 워터마크 제거 여부

# URL 조합
url = f"https://image.pollinations.ai/prompt/{encoded_prompt}?width={width}&height={height}&seed={seed}&model={model}&nologo={nologo}"

response = requests.get(url)
if response.status_code == 200:
    with open("custom_output.png", "wb") as f:
        f.write(response.content)
    print("옵션 적용 이미지 저장 완료!")

```

## 💡 웹 브라우저에서 바로 사용하기

코드 작성 없이 웹 브라우저 주소창에 아래처럼 주소를 직접 입력해도 이미지가 즉시 생성됩니다.

```text
https://image.pollinations.ai/prompt/a%20cute%20cat%20in%20space

```

* **무료/무제한:** API Rate Limit 걱정 없이 테스트나 토이 프로젝트용으로 활용하기 매우 좋습니다.
* **웹앱/GUI 연동:** Streamlit이나 Gradio 같은 파이썬 웹 프레임워크와 결합하면 초간단 AI 이미지 생성기 앱을 금방 제작할 수 있습니다.
