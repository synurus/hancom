02 Hello world!

핵심 개념

HTML은 구조, CSS는 스타일, JavaScript는 상호작용을 담당한다
버튼을 클릭하면 제목 글자가 바뀌는 예제로 이 원리를 보여준다

코드 패턴

const title = document.querySelector("#title");
const btn = document.querySelector("#btn");
btn.addEventListener("click", () => {
  title.textContent = "Hello world!";
});

주요 개념 설명

• const — 변하지 않는 값을 담는 상자를 선언
• document.querySelector() — id로 페이지 요소를 찾는 메서드
• () => { } — 화살표 함수, 이름 없는 함수를 만드는 최신 문법
• addEventListener("click", 함수) — 클릭 시 실행할 코드를 등록하는 이벤트 리스너
• textContent — 요소의 텍스트를 읽거나 바꾸는 속성

파일 구성

HTML에 id 속성으로 구조를 만들고
CSS는 같은 id 선택자로 스타일을 입히고
JavaScript는 같은 id 문자열로 querySelector를 이용해 요소에 접근한다

CSS는 <head> 안에, JavaScript는 </body> 직전에 두는 것이
성능과 요소 인식(로딩 순서) 측면에서 권장되는 방식이다

03 변수 (Variables)

핵심 개념

변수는 이름표 붙은 상자, 그 안에 숫자·글자 같은 값을 넣어두고 필요할 때 꺼내 쓰기
이름을 입력하고 버튼을 누르면 입력한 값을 변수에 담아 화면에 인사말로 보여주는 예제

코드 패턴

const nameInput = document.querySelector("#name");
const out = document.querySelector("#out");

document.querySelector("#greet").addEventListener("click", () => {
  let myName = nameInput.value;
  out.textContent = `안녕, ${myName}!`;
});

주요 개념 설명

• let — 나중에 값이 바뀔 수 있는 상자를 선언 (const는 값이 고정된 상자)
• .value — 입력창(input)에 사용자가 적은 글자를 꺼내오는 속성
• 템플릿 리터럴 `${ }` — 백틱(`) 문자열 안에 변수 값을 끼워 넣는 최신 문법 (옛날 + 이어붙이기 대신)
• = 는 값을 담는 대입 연산자, === 는 값이 같은지 비교하는 연산자로 서로 다름

파일 구성

HTML — input#name, button#greet, p#out 요소로 구조를 만들고
JavaScript — querySelector로 각 요소를 찾아 버튼 클릭 시 입력값을 변수에 저장 후 문장으로 출력
